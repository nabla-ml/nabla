# add

## Signature

```python
endia.add(arg0, arg1) -> 'Array'
```

## Description

Element-wise addition of two arrays or array and scalar.

## Examples

```python
import endia as nb

# Element-wise addition
a = nd.array([1, 2, 3])
b = nd.array([4, 5, 6])
result = nd.add(a, b)
print(result)  # [5, 7, 9]
```
