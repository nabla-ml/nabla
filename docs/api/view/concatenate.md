# concatenate

## Signature

```python
nabla.concatenate(args: 'list[Array]', axis: 'int') -> 'Array'
```

## Description

Concatenate arrays along an existing axis.

## Parameters

- **`args`** (`List of arrays to concatenate`): axis: Axis along which to concatenate arrays (default: 0)

- `Concatenated array`:
