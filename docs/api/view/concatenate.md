# concatenate

## Signature

```python
nabla.concatenate(args: 'list[Tensor]', axis: 'int') -> 'Tensor'
```

## Description

Concatenate tensors along an existing axis.

## Parameters

- **`args`** (`List of tensors to concatenate`): axis: Axis along which to concatenate tensors (default: 0)
