# mul

## Signature

```python
endia.mul(arg0, arg1) -> 'Array'
```

## Description

Element-wise multiplication of two arrays or array and scalar.

## Examples

```python
import endia as nb

# Element-wise multiplication
a = nd.array([1, 2, 3])
b = nd.array([4, 5, 6])
result = nd.mul(a, b)
print(result)  # [4, 10, 18]
```
